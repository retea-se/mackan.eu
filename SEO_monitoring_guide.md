# Google Search Console - SEO Monitoring Guide

## Uppsättning och övervakning efter SEO-förbättringar

### 1. Verifiera webbplats i Google Search Console
- Gå till: https://search.google.com/search-console/
- Lägg till egendom: `mackan.eu`
- Välj verifieringsmetod (HTML-fil eller DNS)

### 2. Skicka in sitemap
```
URL att skicka in: https://mackan.eu/sitemap.php
```

### 3. Viktiga mätvärden att övervaka (veckovis)

#### Indexering (Coverage)
- **Före förbättringar**: ~10-15 indexerade sidor
- **Förväntad förbättring**: 50+ sidor inom 2-4 veckor
- **Kontrollera**: Inga fel eller varningar för viktiga sidor

#### Prestanda (Performance)
**Nyckelord att övervaka:**
- `koordinatkonvertering`
- `tankvolym kalkylator`
- `reservkraftverk dimensionering`
- `QR-kod generator`
- `lösenordsgenerator`
- `RKA kalkylator`

**Mätvärden:**
- Klick: Förväntat ökning 20-40%
- Visningar: Förväntat ökning 50-100%
- CTR: Förväntat förbättring med strukturerad data
- Position: Förväntat förbättring för long-tail keywords

#### Förbättringar (Enhancements)
- **Breadcrumbs**: Kontrollera att de visas korrekt
- **FAQ Rich Results**: Övervaka Featured Snippets
- **Organization markup**: Kontrollera företagspanel

### 4. Specifika kontroller

#### Vecka 1
- [ ] Sitemap indexerad utan fel
- [ ] Breadcrumbs visas i sökresultat
- [ ] Inga kritiska Core Web Vitals-problem

#### Vecka 2-3
- [ ] Ökning av indexerade sidor (målsättning: 40+ sidor)
- [ ] Förbättrade positioner för nyckelord
- [ ] FAQ rich results börjar visas

#### Vecka 4+
- [ ] Stabil ökning av organisk trafik
- [ ] Featured snippets för FAQ-innehåll
- [ ] Förbättrade click-through rates

### 5. Felsökning vanliga problem

#### Om sidor inte indexeras:
1. Kontrollera robots.txt: `https://mackan.eu/robots.txt`
2. Manuell indexering via GSC: "Request indexing"
3. Kontrollera interna länkar

#### Om breadcrumbs inte visas:
1. Validera markup med Rich Results Test
2. Kontrollera att strukturen är korrekt
3. Vänta 2-4 veckor för full implementering

#### Om Featured Snippets uteblir:
1. Förbättra FAQ-format
2. Optimera svarslängd (40-60 ord)
3. Lägg till fler relevanta frågor

### 6. Månatlig rapportering

**KPI:er att spåra:**
- Totala klick från organisk sökning
- Genomsnittlig position för målnyckelord
- Antal indexerade sidor
- Rich results-visningar
- Core Web Vitals-poäng

**Förväntade förbättringar efter 3 månader:**
- +50% organisk trafik
- +30% CTR från sökresultat
- Top 5-positioner för long-tail keywords
- 80%+ av sidor indexerade

### 7. Fortsatta förbättringar

#### Månad 2-3:
- Optimera Core Web Vitals
- Lägg till fler FAQ-sektioner
- Förbättra internal linking
- A/B-testa meta descriptions

#### Månad 4+:
- Skapa innehållskalender
- Optimera för röstökning
- Expandera till fler språk
- Utveckla backlink-strategi
