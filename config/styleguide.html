Styleguide för Mackan.eu

1. Klassnamn och struktur (BEM)

  • Block: komponentens namn, t.ex. .knapp, .kort, .tabell
  • Element: skrivs som __element, t.ex. .kort__rubrik
  • Modifierare: skrivs som --modifierare, t.ex. .knapp--fara

2. Kodexempel

<button class="knapp knapp--stor">Spara</button>

<article class="kort kort--fokus">
  <div class="kort__rubrik">Rubrik</div>
  <div class="kort__innehall">Beskrivning eller innehåll.</div>
</article>

<table class="tabell tabell--kompakt">
  <thead><tr><th>Rubrik</th></tr></thead>
  <tbody><tr><td data-label="Rubrik">Innehåll</td></tr></tbody>
</table>

3. Komponent: Kort

Visuell layoutbehållare med rubrik och innehåll. Används t.ex. i readme eller resultatpaneler.

Rubrik i kort
Detta är innehållet i ett kort. Använd .kort__rubrik och .kort__innehall.
  • .kort – huvudblocket
  • .kort--fokus – visuell förstärkning
  • .kort__rubrik – rubriktext
  • .kort__innehall – brödtext eller innehåll

4. Färg och design tokens

Använd alltid CSS-variabler från variables.css:

:root {
  --primary-color: #007acc;
  --border-radius: 6px;
}

5. Versionshantering

  • Varje CSS-fil ska ha en versionskommentar överst: /* knapp.css - v2 */
  • Vid ändring: öka versionsnummer och notera ändringen i kommentar.

6. AI-regler

  • Använd endast definierade klassnamn från /css/blocks
  • Skapa inga egna klasser utan instruktion
  • Följ BEM-struktur
  • Följ den uppdaterade main.css och översättningstabellen
