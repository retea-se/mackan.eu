# QR v4 – Rapporter

- `lighthouse/`: JSON + HTML från Lighthouse. Namnges `lighthouse-YYYYMMDD-HHMM.json`.
- `seo/`: resultat från SEO-script (t.ex. `seo-YYYYMMDD.json`).
- `aiseo/`: AISEO-insikter (CSV/JSON).
- `smoke/` & `e2e/` skapas automatiskt av testkörningar.

Alla rapporter ska ligga kvar i denna struktur för spårbarhet.
